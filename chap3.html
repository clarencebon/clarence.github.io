WEEK 3: Database Set-up Using XAMPP

XAMPP

is an open source package that is widely used for PHP development. XAMPP contains MariaDB, PHP, and Perl; it provides a graphical interface for SQL (phpMyAdmin), making it easy to maintain data in a relational database.


Basic Steps and guide using XAMPP


Accessing the phpMyAdmin

Open a web browser then enter the URL "http://localhost" to access XAMPP dashboard, after that, select the phpMyAdmin tab.


Other way of accessing:

Alternatively, you may access phpMyAdmin via the XAMPP manager / controller, click Go to Application button to access XAMPP dashboard.

Adding User Account

Step 1: On the phpMyAdmin screen, select User accounts tab. Select Add user account link.

Step 2: Enter user name and password of your choice.

Note: do not use any of your official accounts such as UVA account.

Step 3: Change Host name to "Local."

Step 4: Under the Database for user account, select all the checkboxes below.

Step 5: In the "Global Privileges" section, click the checkbox "Check All."

Step 6: At the bottom-right of the screen click the "Go" button.

Note: Do not change or update the root account. If you may forget or need to reset your password, you can use the root account to manage users.

To verify that the account has been created, go to User Accounts tab. You should see the newly created user account (as shown below).


Creating a Database

Let's create a guestbook database. To create a database, there are several options. You may use the Create database feature.

Step 1: On the phpMyAdmin screen, select the Databases tab. Alternatively, you may click the New link on the left panel.

Step 2: Under the Create database, enter a Database name.

Step 3: Click the Create button.



Another Way of Creating a Database

You may also run the SQL command to create a database.

Step 1: On the phpMyAdmin screen, select the SQL tab.

Step 2: Enter the following syntax "CREATE DATABASE guestbook;"

Note: SQL commands are not case sensitive.

This example uses uppercase and lowercase simply to make it easy to read.

Step 3: Click the "Go" button to run the command.


Note: For Mac users, you may press "Control+Enter" to run.

Note: For Windows users, you may press "Control+Enter" to run.

Creating a Table

Let's create a table named entries.

To create a table, there are several options. You may use the Create table feature.

Step 1: On the phpMyAdmin screen (sidebar), select the "guestbook" database then select the "Structure" tab.

Step 2: Under the Create table, enter a table name and the number of columns you want.

Step 3: Click the "Go" button, this will prompt you to enter the column information.
